# Development Commands

Create commits that are useful from a changelog perspective.

Use the Taskfile commands for common development work:

- `task run` to run Kivra.
- `task build` to build the release executable.
- `task test` to run tests.
- `task format` to format Swift source files.
- `task lint` to run formatting and SwiftLint checks.
- `task install-dev` to build and install Kivra Dev.

Run `task` to list available commands.
